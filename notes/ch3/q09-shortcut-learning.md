# q09 — 모델이 배경(글씨·접시)을 보고 행동을 맞히면, 정확도는 높은데 왜 문제인가?

관련: [ch3 concepts — shortcut learning](concepts.md#지름길-학습--거짓-상관-shortcut-learning--spurious-correlation),
[ch3 notes — 진단 결과](notes.md#grad-cam--확신하며-틀린-3장이-같은-병을-가리킴) · `scripts/gradcam.py`

## 무슨 상황이었나

exp02 Grad-CAM 진단에서, 확신 98~100%로 틀린 3장이 전부 **사람의 행동이 아니라
배경**을 보고 있었다:

- `phoning_253`: 전화기가 아니라 **뒤 벽의 글씨**를 보고 → writing_on_a_board (99%)
- `reading_076`: 손의 책이 아니라 **뒤 칠판 글씨**를 보고 → writing_on_a_board (100%)
- `watching_TV_017`: TV가 아니라 **앞 식탁의 접시**를 보고 → cooking (98%)

그런데 이런 사진들에서도 **train 정확도는 잘 나온다.** "글씨 있는 판이 보이면
writing_on_a_board"가 학습 데이터 안에서는 자주 맞으니까. 그럼 뭐가 문제인가?

## 스스로 생각해보기

- 학습 데이터에서 "칠판 = 글씨 쓰기"가 90% 맞다면, 모델이 그 규칙을 쓰는 게
  왜 나쁜가? 시험 점수는 오르는데?
- 이 모델을 실제로 배포한다면(예: 사진 앱의 자동 태깅), 어떤 사진에서 처음
  크게 틀릴까?
- 테슬라 자율주행이 이와 똑같은 실수를 한다면 어떤 모습일까?

### 핵심: 정확도는 "지금 이 데이터에서"의 성적일 뿐

정확도가 재는 건 **"이 test set에서 몇 개 맞혔나"**이지 **"올바른 이유로
맞혔나"**가 아니다. 모델이 배경으로 찍어도, test set 사진들 배경이 학습 때와
비슷하면 점수는 나온다. 즉 **높은 정확도가 "제대로 이해했다"를 보장하지 않는다.**

문제는 그 지름길이 **의존하는 상관이 깨지는 순간** 드러난다:

1. **분포가 바뀌면 무너진다.** "칠판 = 쓰기" 지름길은, 칠판 앞에서 *책 읽는*
   사람(reading_076)을 만나는 순간 100% 확신으로 틀린다. 배경이 학습 때와
   달라지는 실제 배포 환경이 정확히 이 상황이다 → [분포 변화](../ch1/concepts.md)
   (ch1에서 배운 train/test 분리의 더 깊은 버전).

2. **근거가 틀리면 못 믿는다.** 자동 태깅 앱이 "카페에서 책 읽는 사진"을
   "칠판 있으니 필기 중"으로 태깅하면, 정확도 숫자와 무관하게 **신뢰를 잃는다.**
   사람은 "왜 그렇게 판단했어?"를 묻는데, 근거가 배경이면 답이 궁색하다.

3. **개선 방향을 오도한다.** 배경으로 맞히는 모델에 데이터를 더 부어도, 같은
   지름길을 더 굳게 배울 뿐이다. **무엇을 보고 맞히는지**를 봐야(=Grad-CAM)
   "증강으로 배경을 흔들자" 같은 옳은 처방이 나온다.

### 테슬라로 옮기면

- 초기 차선 인식이 "차선 = 밝은 흰 선"이라는 지름길을 배우면, 맑은 날 고속도로
  데이터에선 99% 정확하다. 하지만 **눈 덮인 길, 색 바랜 선, 공사장 임시선**에서
  갑자기 무너진다 — 상관이 깨지는 분포로 넘어갔기 때문.
- 더 위험한 버전: 데이터에 우연히 얽힌 단서. "이 시간대 이 도로엔 늘 앞차가
  있었다 → 앞차 그림자를 차선 신호로 오해" 같은. 정확도로는 안 보이고,
  **어디를 보고 반응했나(히트맵)**를 봐야 잡힌다.
- 그래서 테슬라는 ① 다양한 환경 데이터로 지름길을 못 쓰게 하고(→ 우리 증강),
  ② 개입 순간의 인식 근거를 뜯어본다(→ 우리 Grad-CAM). 우리가 ch3에서 하는
  작업의 대형 버전이다.

### 한 줄 요약

> **정확도는 "맞혔나", Grad-CAM은 "맞는 이유로 맞혔나"를 잰다.**
> 배포되는 순간 중요한 건 후자다 — 배경은 우리가 통제할 수 없으니까.

맞는데(사람·손을 봄) 마지막 구분에서 지는" 다른 종류의 오답 — 지름길과 구별할 것.
